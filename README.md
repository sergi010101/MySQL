# 🐬 MySQL 

Проверил функционал (E2E) процесса покупки: от регистрации до оплаты, с акцентом на проверку корректности записи данных в базу MySQL.
1) Тестирвание регистрации:
 <ul>
 <li>Создал нового пользователя через API</li> 
 <li>Выполнил SQL-запросы для проверки, что все поля (хеш пароля, дата создания, логин и пароль) корректно сохранились в базе данных </li> 
 </ul>
 2) Тестирование корзины и заказа: 
 <ul>
 <li>Добавил товары в корзину и оформил заказ</li>
 <li>Проверил в БД, что заказ привязался к верному id-пользователя, а итоговая сумма совпадает с ценой товара</li>
 </ul>
 3) Тестирование оплаты:
 <ul>
 <li>Проверил оплату заказа, используя тестовые карты</li>
 <li>Проверил критичеси важные изменения (обновление статуса заказа, оплату заказа, совпадение суммы транзакции)</li>
 </ul>
